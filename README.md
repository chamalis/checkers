--------------------------------------------------

Compilation: 

* make [all]  - to build client and server
* make client - to build just the client
* make server - to build just the server

Execution: 

./guiServer
./server [-p port] [-g number_of_games] [-s (swap color after each game)]
./client [-i ip] [-p port]
